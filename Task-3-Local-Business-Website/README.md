
# 🚛 Task 3 - Local Business Website & Live Pitch Project

## 📌 Project Title
**M.S. Yadav Transport Service Website**

## 📖 Project Overview
This project is a professional website developed for **M.S. Yadav Transport Service**, a transportation and logistics company. The website is designed to strengthen the company's online presence, improve customer communication, and make transport booking easier through a modern and responsive web interface.

---

## ✨ Features

- Responsive Design
- Home Page
- About Us
- Services Section
- Fleet Information
- Book Transport Form
- Contact Page
- Call Now Button
- Mobile Friendly Layout
- SEO-Friendly Structure

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- React.js
- Vite

---

## 🎯 Objectives

- Create a professional digital identity for the business.
- Improve customer engagement.
- Simplify transport booking.
- Increase online visibility.
- Build customer trust.

---

## 🚀 Business Benefits

The website helps the business by:

- Increasing online visibility.
- Building trust with new customers.
- Allowing customers to book transport services easily.
- Providing quick communication through the Call Now feature.
- Showcasing the company's fleet and services professionally.
- Supporting business growth through a strong digital presence.
